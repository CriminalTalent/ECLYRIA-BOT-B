## 기본 전투 룰

### 1:1 전투

#### 시작 조건
- 2명의 플레이어 필요
- 둘 다 전투 중이 아니어야 함
- 명령어: `[전투개시/@상대]`

#### 선공 결정
```
민첩 스탯 + D20 굴림
높은 쪽이 선공
```

**예시:**
```
Player1: 민첩 10 + D20(15) = 25
Player2: 민첩 12 + D20(10) = 22
→ Player1 선공
```

#### 턴 진행
1. 현재 턴 플레이어가 행동 선택
   - 공격
   - 방어
   - 반격
   - 물약 사용
   - 도주
2. 행동 처리
3. 다음 턴으로 이동

#### 공격 판정
```
공격자 공격력 + D20
vs
방어자 방어력 + D20

데미지 = (공격 합계) - (방어 합계)
최소 데미지 = 0
```

**예시:**
```
공격자: 공격 15 + D20(12) = 27
방어자: 방어 10 + D20(8) = 18
데미지: 27 - 18 = 9
```

#### 치명타
```
확률 = (행운 스탯 / 2)% (최대 50%)
효과 = 데미지 * 1.5
```

**예시:**
```
행운 20 → 확률 10%
D100 굴림: 7 → 치명타 발동!
기본 데미지 10 → 치명타 15
```

#### 방어 태세
- 선언 턴: 방어 태세 돌입
- 다음 피격 시: 방어 주사위 2회 판정
- 높은 쪽 사용

**예시:**
```
방어 태세 선언
→ 다음 턴 공격 받음
→ 방어 D20 2회: 15, 8
→ 15 사용
```

#### 반격 태세
- 선언 턴: 반격 태세 돌입
- 다음 피격 시: 고정 5 데미지 반격
- 1회 소모

#### 도주
```
판정: D20 + 행운 + 민첩
난이도: 25

성공 → 전투 종료
실패 → 턴 소비
```

#### 승리 조건
- 상대 HP 0 이하
- 상대 도주 성공

### 2:2 팀 전투

#### 시작 조건
- 4명의 플레이어 필요
- 팀1 (2명) vs 팀2 (2명)
- 명령어: `[다인전투/@A/@B/@C/@D]`
  - A, B = 팀1
  - C, D = 팀2

#### 선공 결정
```
팀1 민첩 합계 + D20
vs
팀2 민첩 합계 + D20

높은 팀이 선공
```

**예시:**
```
팀1: (민첩10 + 민첩12) + D20(15) = 37
팀2: (민첩8 + 민첩14) + D20(12) = 34
→ 팀1 선공
```

#### 턴 순서
```
선공 팀 순서:
1. 팀 내 민첩 높은 플레이어
2. 팀 내 민첩 낮은 플레이어

후공 팀 순서:
3. 팀 내 민첩 높은 플레이어
4. 팀 내 민첩 낮은 플레이어
```

#### 라운드 진행
1. 4명 모두 행동 선택 (순서대로)
2. 모든 행동 동시 처리
3. HP 확인
4. 다음 라운드

#### 타겟 지정
```
[공격/@타겟]    # 특정 적 공격
[방어/@타겟]    # 특정 아군 보호
```

**제한:**
- 아군은 공격 불가
- 적은 방어 불가

#### 승리 조건
- 상대 팀 전원 HP 0 이하

## 공동목표 룰

### 시작 조건
- 참가 인원: 1~30명
- 레이드: 최소 3명
- 층 선택: B2, B3, B4, B5

### 층별 정보
| 층 | 이름 | 난이도 | 적 종류 | 조사 유형 |
|----|------|--------|---------|-----------|
| B2 | 지하 2층 | 1 | 활동가, 지지자 | 조사 |
| B3 | 지하 3층 | 2 | 지지자, 집행자 | 정밀조사 |
| B4 | 지하 4층 | 3 | 집행자, 간부, 정예 | 감지 |
| B5 | 지하 5층 | 4 | 간부, 정예, 사령관 | 훔쳐보기 |

### 맵 시스템
```
8x8 그리드 맵
참가자: 하단(y=6-7) 배치
적: 상단(y=0-2) 배치
```

### 이동 규칙
- 한 번에 한 칸 (8방향)
- 점유된 칸 이동 불가
- 맵 밖 이동 불가

### 조사 판정 (이동 시 자동)
```
1. 위치 확률 체크
   - 상단(0-2): 50%
   - 중단(3-5): 30%
   - 하단(6-7): 15%

2. 확률 통과 시 조사 판정
   D20 + 행운 >= 난이도

3. 성공 시 단서 발견
   - 결과 텍스트 표시
   - 아이템/갈레온 지급
   - 로그 기록
   - 맵에 ? 표시

4. 같은 위치 중복 조사 불가
```

**예시:**
```
Player 이동: (0, 6) → (0, 5)
위치: 중단 → 확률 30%
D100 굴림: 25 → 통과!

조사 판정:
D20: 15
행운: 12
합계: 27
난이도: 20
→ 성공!

결과: "집행자의 일지를 발견했습니다."
보상: [아이템:일지] [갈레온:50]
```

### 전투 규칙
- 적과 인접 시 공격 가능
- 1:1 전투와 동일한 판정
- 여러 명이 동시 공격 가능

### 적 종류
| 적 | HP | 공격 | 방어 | 민첩 | 행운 | 경험치 |
|----|-----|------|------|------|------|--------|
| 순혈주의 활동가 | 40 | 3 | 2 | 3 | 5 | 10 |
| 클라리스 지지자 | 50 | 4 | 3 | 4 | 6 | 15 |
| 혈통차별 집행자 | 70 | 5 | 4 | 5 | 8 | 25 |
| 클라리스 간부 | 90 | 6 | 5 | 6 | 10 | 35 |
| 정예 순혈주의자 | 120 | 8 | 6 | 7 | 12 | 50 |
| 클라리스 사령관 | 150 | 10 | 8 | 8 | 15 | 75 |

### 레이드 보스
```
클라리스 오르이 핵심인물
HP: 300
공격: 12
방어: 10
민첩: 10
행운: 20
경험치: 200

특수 능력:
- 멀티 어택: 한 턴에 3명 공격
- 타겟 우선순위: 보스 공격한 플레이어
- 데미지 감소: 플레이어 데미지 30% 감소
```

**멀티 어택 예시:**
```
Player1이 보스 공격
→ 보스 턴
→ 공격 대상 선정 (최대 3명)
  1. Player1 (우선 타겟)
  2. Player2 (랜덤)
  3. Player3 (랜덤)
→ 각각 공격 판정
```

### 완료 조건
- 모든 적 처치
- 발견한 단서 집계

## 조사 시스템 룰

### 조사 시작
```
[조사시작]
→ 위치 선택 가능
→ 조사, 세부조사, 이동 가능
```

### 위치 조사
```
[조사/위치]
→ 위치 개요 출력
→ 세부 조사 대상 목록
→ 자동 판정 (있는 경우)
```

### 세부 조사
```
[세부조사/대상]
→ 조사 판정 (D20 + 행운)
→ 성공/실패 결과
→ 보상 지급
→ 로그 기록
```

### 이동 제한
```
하루 3회 제한
자정 초기화
```

### 협력 조사
```
조건:
1. 같은 위치에 있어야 함
2. 둘 다 조사 중이어야 함

효과:
- 행운 합산 + 보너스(+5)
- 판정 유리
- 보상 각자 지급
```

**예시:**
```
Player1: 행운 10
Player2: 행운 12
협력 보너스: +5
합계: 27

D20 굴림: 15
최종: 42
난이도: 30
→ 성공!
```

### 방해
```
효과:
- 대상의 다음 판정에 -3
- 방해 성공 자동 (판정 없음)
```

### 조사 종류
| 종류 | 설명 | 비고 |
|------|------|------|
| 조사 | 기본 조사 | 가장 쉬움 |
| 정밀조사 | 상세 조사 | 중간 난이도 |
| 감지 | 마법/기운 감지 | 높은 난이도 |
| 훔쳐보기 | 은밀한 관찰 | 매우 높은 난이도 |
| DM조사 | DM 전용 | 특수 |

## 스탯 시스템

### 기본 스탯
| 스탯 | 설명 | 기본값 | 영향 |
|------|------|--------|------|
| HP | 체력 | 100 | 생존 |
| 공격 | 공격력 | 10 | 데미지 증가 |
| 방어 | 방어력 | 10 | 데미지 감소 |
| 민첩 | 민첩성 | 10 | 선공, 도주 |
| 행운 | 행운 | 10 | 치명타, 조사 |

### 스탯 효과

#### 공격
```
기본 데미지 = 공격 스탯 + D20
```

#### 방어
```
데미지 감소 = 방어 스탯 + D20
```

#### 민첩
```
선공 판정 = 민첩 + D20
도주 판정 = D20 + 행운 + 민첩
```

#### 행운
```
치명타 확률 = 행운 / 2 (최대 50%)
조사 판정 = D20 + 행운
```

| 전사 | 100 | 15 | 12 | 8 | 5 |
| 도적 | 100 | 12 | 8 | 15 | 10 |
| 마법사 | 100 | 10 | 6 | 10 | 14 |
| 탱커 | 100 | 8 | 15 | 7 | 5 |
| 균형형 | 100 | 10 | 10 | 10 | 10 |

## 판정 시스템

### D20 시스템
```
기본 판정 = D20 + 스탯
대결 판정 = (D20 + 스탯) vs (D20 + 스탯)
```

### 판정 종류

#### 공격 판정
```
공격자: 공격 + D20
방어자: 방어 + D20
데미지: MAX(0, 공격 합계 - 방어 합계)
```

#### 조사 판정
```
D20 + 행운 >= 난이도
성공: 성공 결과 출력
실패: 실패 결과 출력
```

#### 도주 판정
```
D20 + 행운 + 민첩 >= 25
성공: 전투 종료
실패: 턴 소비
```

#### 치명타 판정
```
D100 <= (행운 / 2)
성공: 데미지 * 1.5
실패: 일반 데미지
```

### 난이도 기준
| 난이도 | 설명 | 예시 |
|--------|------|------|
| 5-10 | 매우 쉬움 | 기본 조사 |
| 11-15 | 쉬움 | 일반 조사 |
| 16-20 | 보통 | 정밀조사 |
| 21-25 | 어려움 | 감지, 도주 |
| 26-30 | 매우 어려움 | 훔쳐보기 |
| 31+ | 거의 불가능 | 특수 |

## 물약 및 아이템

### 물약 종류
| 물약 | 회복량 | 가격 | 설명 |
|------|--------|------|------|
| 소형 물약 | 10 | - | 기본 회복 |
| 중형 물약 | 30 | - | 중간 회복 |
| 대형 물약 | 50 | - | 큰 회복 |
| 물약 | 20 | - | 일반 물약 |

### 물약 사용 규칙
```
전투 중:
- 자신의 턴에만 사용 가능
- 턴 소비
- 대상 지정 가능 (2:2)

전투 외:
- 언제든지 사용 가능
- 최대 HP(100) 초과 불가
```

### 아이템 획득
```
조사 성공 시:
[아이템:아이템명] 태그 → 자동 지급

예시:
"비밀 문서 발견! [아이템:비밀문서]"
→ "비밀문서" 아이템 인벤토리 추가
```

### 갈레온 획득
```
조사 성공 시:
[갈레온:숫자] 태그 → 자동 지급

예시:
"보물 상자 발견! [갈레온:100]"
→ 갈레온 100 추가
```

## 제한 및 규칙

### 중복 불가
- 한 사람은 하나의 전투만
- 한 사람은 하나의 공동목표만
- 같은 위치 중복 조사 불가

### 동시 진행 가능
- 전투 + 조사
- 여러 공동목표 (다른 팀)
- 여러 전투 (다른 사람들)

### 턴 제한
- 전투: 순서대로
- 공동목표: 자유롭게
- 조사 이동: 하루 3회

### 인원 제한
| 시스템 | 최소 | 최대 |
|--------|------|------|
| 1:1 전투 | 2 | 2 |
| 2:2 전투 | 4 | 4 |
| 허수아비 | 1 | 1 |
| 공동목표 | 1 | 30 |
| 레이드 | 3 | 30 |

### 시간 제한
- 조사 이동: 하루 3회 (자정 초기화)
- 전투: 없음
- 공동목표: 없음

## 팁 및 전략

### 전투 팁
1. 높은 민첩 = 선공 유리
2. 방어 태세 = 큰 공격 대비
3. 반격 태세 = 공격적 플레이
4. 도주 = 행운+민첩 필요

### 공동목표 팁
1. 대각선 이동 = 빠른 접근
2. 상단 집중 = 단서 많음
3. 분산 탐색 = 효율적
4. 행운 스탯 = 단서 발견 유리

### 조사 팁
1. 협력 조사 = 판정 유리
2. 방해 활용 = 경쟁 시
3. 이동 3회 = 계획적 사용
