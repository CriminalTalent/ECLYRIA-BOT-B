# 조사 연동 공동목표 시스템 가이드

## 새로운 기능: 이동하며 단서 발견! 🔍

### 개요
공동목표를 진행하면서 **이동할 때마다** 자동으로 조사 판정이 이루어집니다.
- 조사 시트와 완전 연동
- 층별로 다른 조사 유형 적용
- 이동 중 단서 자동 발견
- 발견한 단서는 맵에 표시 (?)

## 층별 조사 유형

| 층 | 조사 유형 | 난이도 | 발견 확률 |
|----|----------|--------|----------|
| B2 | 조사 | 낮음 | 상단 50% |
| B3 | 정밀조사 | 중간 | 상단 50% |
| B4 | 감지 | 높음 | 상단 50% |
| B5 | 훔쳐보기 | 최고 | 상단 50% |

### 위치별 발견 확률
- **상단 (0~2행)**: 50% - 적 근처라 단서가 많음
- **중단 (3~5행)**: 30% - 보통
- **하단 (6~7행)**: 15% - 시작 위치라 단서 적음

## 조사 시트 설정

### 필수 설정
조사 시트에 다음 항목을 추가하세요:

```
대상              | 종류      | 난이도 | 성공결과                           | 실패결과
지하 2층 단서     | 조사      | 10     | 활동가들의 은신처 위치를 발견...    | 희미한 흔적만 보입니다
지하 3층 단서     | 정밀조사  | 15     | 집행자의 작전 계획서를 발견...      | 중요한 정보는 없습니다
지하 4층 단서     | 감지      | 20     | 강력한 마법의 흔적을 감지...        | 특별한 것을 느끼지 못했습니다
지하 5층 단서     | 훔쳐보기  | 25     | 핵심인물의 비밀 문서를 발견...      | 경계가 너무 삼엄합니다
```

### 보상 추가 (선택)
```
대상              | 종류      | 난이도 | 성공결과
지하 3층 단서     | 정밀조사  | 15     | 비밀 문서 발견! [아이템:비밀문서] [갈레온:50]
```

```

→ 발견한 단서는 공유됨
→ 맵에 ? 표시로 누적

공동목표 완료!
발견한 단서 총 8개
```

## 전략 팁

### 단서 수집 전략
1. **상단 집중**: 적 근처(0~2행)가 단서가 많음
2. **대각선 이동**: 빠르게 상단 도달
3. **분산 탐색**: 여러 명이 다른 경로로 이동
4. **행운 스탯**: 행운 높으면 판정 유리

### 효율적인 플레이
```
# 나쁜 예: 직선 이동
[이동/상] → [이동/상] → [이동/상]
(같은 경로 = 중복 확률 높음)

# 좋은 예: 지그재그 이동
[이동/상] → [이동/우상] → [이동/상] → [이동/좌상]
(다양한 위치 = 더 많은 단서 발견)
```

### 팀 협력
- **탐색팀**: 행운 높은 플레이어가 먼저 이동
- **전투팀**: 공격 높은 플레이어가 적 처치
- **정보 공유**: 발견한 단서는 모두 공유됨

## 조사 판정 시스템

### 판정 공식
```
행운 스탯 + D20 >= 난이도
```

### 예시
```
플레이어: 행운 12
D20 굴림: 15
합계: 27
난이도: 15
→ 성공!
```

### 단서 발견 조건
1. **위치 확률** 통과 (상단 50%, 중단 30%, 하단 15%)
2. **조사 판정** 성공 (행운 + D20 >= 난이도)
3. **중복 방지** (같은 위치는 한 번만)

## 조사 로그 기록

모든 단서 발견은 자동으로 조사로그 시트에 기록됩니다:
```
시간              | 사용자 | 위치      | 대상           | 종류      | 결과 | 내용
2025-12-01 10:30 | Alice  | 지하 3층  | 지하 3층 단서  | 정밀조사  | 성공 | 집행자의 작전 계획서...
```

## 보상 시스템

### 자동 지급
조사 성공 시 `[아이템:아이템명]`, `[갈레온:숫자]` 태그가 있으면 자동 지급:

```
성공결과: "비밀 문서 발견! [아이템:비밀문서] [갈레온:100]"
→ 자동으로 아이템과 갈레온 지급
```

## 명령어

### 기본 명령어 (변경 없음)
```
[공동목표/B3/@Players]  # 시작
[이동/방향]             # 이동 + 자동 조사
[목표공격]              # 공격
[맵보기]                # 맵 확인 (? 단서 표시)
[목표상태]              # 상태 (단서 개수 포함)
[목표포기]              # 포기
```

## 장점

### 1. 자연스러운 통합
- 별도 조사 명령 불필요
- 이동하며 자동으로 발견
- RP 몰입감 증가

### 2. 전략적 요소
- 경로 선택이 중요
- 행운 스탯의 활용
- 팀 협력 필요

### 3. 보상 시스템
- 단서 발견 시 아이템/갈레온
- 조사 로그 자동 기록
- 전투 + 조사 동시 진행

## 주의사항

1. 조사 시트에 층별 단서 설정 필수
2. 같은 위치는 한 번만 조사됨
3. 확률적이므로 모든 칸에서 발견하진 않음
4. 행운 스탯이 낮으면 판정 실패 가능
